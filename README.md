STUDENT PASS/FAIL PREDICTION AI
PROJECT OVERVIEW

This project uses Machine Learning to predict whether a student will Pass or Fail based on:
- Study Hours
- Attendance Percentage

 TECHNOLOGIES USED
- Python
- Pandas
- Scikit-learn

ALGORITHM
Decision Tree Classifier

 FEATURES
- Loads student dataset
- Trains AI model
- Tests model accuracy
- Predicts Pass/Fail for new students
- 
 HOW TO RUN
pip install pandas scikit-learn

python student_prediction.py
